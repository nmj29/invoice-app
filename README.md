# InvoiceGen — M/S Ganapati Trading Co

A fast, AI-assisted **GST Tax Invoice Generator** for M/S Ganapati Trading Co.

## 🌐 Live App
👉 **[Open Invoice Generator](https://YOUR_USERNAME.github.io/invoice-app/)**

## ✨ Features
- Select client → all details auto-fill
- Select product → HSN code, rate, description auto-fill
- Auto invoice numbering with financial year (e.g. `107/2026-27`)
- CGST 2.5% + SGST 2.5% auto-calculated
- Amount in words (Indian Rupees)
- Download pixel-perfect A4 PDF matching the original invoice format
- Add new clients and products — saved permanently in your browser

## 📋 How to Use
1. Open the link above in any browser
2. Select a **Client** from the dropdown (or add a new one with ＋)
3. Add one or more **Line Items** — pick a product or type manually
4. Fill in **Quantity** → totals calculate automatically
5. Click **Download PDF** — A4 invoice is saved to your device
